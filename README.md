tinyI2C
=======

tiny UART-I2C converter project using mbed microcontrollers.

  tinyI2C
  |-gui       <-- PyQt4 based GUI source codes
  |-mbed      <-- mbed source codes
  |-pcb       <-- PCB design files for eagle
  |-python    <-- python module code with standalone test program
